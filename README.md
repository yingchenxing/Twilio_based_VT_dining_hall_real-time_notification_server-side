# Twilio based VT dining hall real-time notification (Server side)
This project is based on Spring Boot framework. 


## Environment
- jdk 1.8
- MySQL 5.7+
- MyBatis 3.5.1
- Lombok
- Hutool 5.7.20
- MyBatis-plus 3.5.1
- twilio 9.0.1
- SpringBoot  2.6.8
